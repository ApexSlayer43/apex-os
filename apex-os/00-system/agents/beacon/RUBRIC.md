# BEACON — Evaluation Rubric

## Criteria

### 1. Message Clarity (30%)
- **5:** One-sentence positioning immediately clear, headline assembled from customer language
- **3:** Positioning mostly clear, some inside-voice language
- **1:** Positioning sounds like marketing — "streamline," "best-in-class," "empower"

### 2. Channel Fit (25%)
- **5:** Specific community named with URL, protocol defined week-by-week, one primary channel committed
- **3:** Channel identified but protocol vague, or multiple channels without clear priority
- **1:** Generic "social media" recommendation, no specific community

### 3. Customer Language Accuracy (25%)
- **5:** All copy assembled from VIGIL's customer quotes, Schwartz test passed on every line
- **3:** Mostly customer language, some founder-voice leaks
- **1:** Copy written from inside the founder's head, no customer language

### 4. Conversion Focus (20%)
- **5:** Clear path to first 10 customers, specific outreach targets identified, evangelist trigger designed
- **3:** General acquisition plan, some specificity
- **1:** Brand awareness focus, no path to paying customers

## Anti-Pattern Indicators (automatic score of 1)
- Used "streamlined workflows," "best-in-class," or similar marketing fluff
- Recommended Product Hunt as primary strategy
- No VOC assembly — copy written from scratch
- Positioned against SaaS competitor when real alternative is "do nothing"
