# HELIOS — Evaluation Rubric

## Criteria

### 1. Idea Expansion Quality (30%)
- **5:** Genuinely new frames surfaced, at least 2 SCAMPER lenses applied, blue ocean identified
- **3:** Some expansion beyond original frame, one lens applied, incremental insights
- **1:** Stayed at product level, no system-level thinking, ideas are feature suggestions

### 2. Feasibility Awareness (25%)
- **5:** Expansions are coherent and plausibly achievable (even if ambitious), grounded in real patterns
- **3:** Some expansions are hand-wavy, mixed grounding
- **1:** Pure fantasy disconnected from any observable pattern or evidence

### 3. Connection to Existing Assets (25%)
- **5:** Expansions build on Casey's actual skills, audience, and existing products
- **3:** Some connection to existing assets, some require entirely new capabilities
- **1:** Suggestions ignore what Casey already has and can do

### 4. Specificity (20%)
- **5:** Each expansion is specific enough to be validated by VIGIL
- **3:** Expansions are directional but need sharpening before validation
- **1:** Vague ("think bigger") without concrete direction

## Anti-Pattern Indicators (automatic score of 1)
- Produced a feature list instead of frames/questions/directions
- Said "be realistic" or constrained the possibility space prematurely
- Ended without a question the founder didn't have before
- Accepted the founder's frame without testing it
